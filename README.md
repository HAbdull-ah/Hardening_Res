# Wayne State University CCDC Team Repository

This repository contains scripts, reference materials, and operational checklists used by the **Wayne State University Collegiate Cyber Defense Competition (CCDC) Team** during training and competition.

The materials included here are intended to support system hardening, service auditing, incident response, and rapid defensive actions under competition conditions.

---

## Repository Contents

- Defensive and hardening scripts  
- Service validation and auditing utilities  
- Competition reference notes  
- “Doomsday” rapid-response checklists  

All materials are maintained for internal team use and continuous improvement between practice sessions and competition rounds.
